# Examples Optional and ReactiveOptional

## When to Use What

**Use `Optional`** (synchronous) when:
- Simple value transformations
- No reactivity needed
- Pure functions / utilities
- One-time operations

[Optional Examples](./EXAMPLES_OPTIONAL.md)

**Use `ReactiveOptional`** when:
- Working with backend DTOs
- Need Vue reactivity
- Async data loading
- Form bindings required
- Composable pattern

[ReactiveOptional Examples](./EXAMPLES_REACTIVE_OPTIONAL.md)
